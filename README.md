## Quick start 🏄

Prerequisites: [Node (v16 LTS)](https://nodejs.org/en/download/) and [Git](https://git-scm.com/downloads)

> Clone the repository:

```
git clone https://github.com/salviega/cyclimate-frontend-react.git
```

> Install dependencies:

```
npm install
```

> Run the project:

```
npm start
```
